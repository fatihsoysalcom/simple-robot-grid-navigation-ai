# Simple Robot Grid Navigation AI

This example demonstrates a basic AI agent (a 'robot') navigating a grid environment. The robot uses simple perception to check adjacent cells and decision-making logic to choose a path that minimizes its distance to a target, avoiding obstacles. It illustrates fundamental concepts of AI in robotics like environmental perception and goal-oriented movement.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
